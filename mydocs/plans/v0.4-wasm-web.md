## 핵심 가치

Rust 코어를 WebAssembly로 컴파일해 브라우저에서 직접 실행한다.  
`@rpdf/core` npm 패키지 + GitHub Pages 데모로 **외부 공개 가능한 첫 버전**.  
v0.5 Tauri UI의 기반이 되는 React 프론트엔드가 이 단계에서 완성된다.

## 핵심 의존성

- **v0.3 편집 커맨드 완료**: WASM API는 편집 커맨드를 그대로 노출
- **wasm-pack**: Rust → WASM 컴파일 도구
- **pdf.js**: 브라우저 렌더링 (pdfium 대체)
- **Web Worker + Comlink**: 파싱/저장 메인 스레드 블록 방지

---

## 성공 기준 (초안)

1. `wasm-pack build` 성공, `pkg/` 생성
2. `@rpdf/core` npm 배포 가능한 형태
3. 브라우저에서 파일 열기 → 썸네일 → 병합 → 다운로드 동작
4. GitHub Pages 데모 사이트 접근 가능
5. WASM 번들 2MB 이하 (gzip)
