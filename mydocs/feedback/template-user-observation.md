## 관찰 방법

1. 기존 pdf-studio와 rpdf를 동시에 설치
2. 평소 PDF 작업을 자연스럽게 진행
3. 옆에서 관찰하며 기록 (또는 본인이 사용 후 공유)
4. 명시적 질문은 최소화 (행동 기반 관찰 우선)
