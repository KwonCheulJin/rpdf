# feedback/

코드 리뷰, PR 셀프 리뷰, 실사용자 피드백 등을 보존하는 디렉터리.

## 파일 명명 규칙

- PR 셀프 리뷰: `task{N}-pr-review.md`
- 코드 리뷰: `task{N}-code-review.md`
- 실사용자 피드백: `user-{기간}.md` (예: `user-2026-Q3.md`)

## 현재 파일 목록

| 파일 | 내용 |
|------|------|
| `task2-pr-review.md` | Task #2 PR 셀프 리뷰 — 7건 발견, 3건 즉시 처리, 4건 백로그 |

## 템플릿

- `template-code-review.md` — 코드 리뷰 작성 가이드
- `template-user-observation.md` — 실사용자 피드백 기록 양식

PR 셀프 리뷰는 별도 템플릿 없이 발견 내용 위주로 자유 작성.
