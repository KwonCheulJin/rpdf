# examples/

rpdf 개발 및 디버깅에 사용하는 샘플 PDF 모음입니다.
모두 **공개 도메인 또는 오픈 라이선스** 파일입니다.

## 파일 목록

| 파일 | 출처 | 특징 | 라이선스 |
| --- | --- | --- | --- |
| `fw4-2024.pdf` | [IRS Form W-4](https://www.irs.gov/pub/irs-pdf/fw4.pdf) | 폼 필드·주석 포함, PDF 1.7 | 미국 정부 공개 도메인 |
| `irs-f1040.pdf` | [IRS Form 1040](https://www.irs.gov/pub/irs-pdf/f1040.pdf) | 복잡한 폼·레이아웃, PDF 1.7 | 미국 정부 공개 도메인 |
| `pdfjs-basicapi.pdf` | [PDF.js 테스트 스위트](https://github.com/mozilla/pdf.js) | 기본 API 테스트용, 3페이지, PDF 1.6 | Apache 2.0 |
| `pdfjs-tracemonkey.pdf` | [PDF.js 테스트 스위트](https://github.com/mozilla/pdf.js) | 6페이지 기술 문서, PDF 1.4 | Apache 2.0 |
| `pdfjs-annotation-border.pdf` | [PDF.js 테스트 스위트](https://github.com/mozilla/pdf.js) | 주석 테두리 스타일, 1페이지, PDF 1.5 | Apache 2.0 |

## 파일 추가 기준

- 1MB 이하 파일만 git에 커밋한다
- 명확한 공개 도메인 또는 오픈 라이선스 확인 필수
- 파일 추가 시 이 README에 출처와 특징을 기록한다
- 특정 버그 재현용 샘플은 `samples/`에, 일반 예시는 `examples/`에 둔다
